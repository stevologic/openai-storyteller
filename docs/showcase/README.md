# Showcase assets

Drop **clean, chrome-free** storybook images here to fill the gallery in the main
[`README.md`](../../README.md). Full-page browser screenshots don't belong here — they
include the browser bar and bookmarks.

## How to get clean images

In the app, open a finished book and, on the closing page, click:

- **Download pictures** → the cover + every page as individual PNGs, or
- **Save file** → a single `.storyteller.json` (I can extract the images from it).

## Suggested filenames

| File | What |
|---|---|
| `hero.png` | A cover to feature at the top of the README |
| `billy-00-cover.png`, `billy-03.png`, `billy-04.png`, `billy-06.png` | Pages of _Billy's Broccoli Path_ |

Any names work — once the files are here, the `<!-- … -->` placeholders in the main README
get swapped for `<img>` tags pointing at them.
